Users can be added in a hierarchy.
When a user makes a sale, commission is distributed up to 5 levels of parent users based on certain fixed percentages.

USERS:
  Users can be added under another user.
  Parent users are selected using a dropdown(no need to know user ID).
  Hierarchy can go deeper than 5 levels, but commission is limited to 5 levels.

Sales:
  Sales are recorded for a selected user(using dropdown).
  Sale amount must be greater than zero.
  After recording a sale, commissions are automatically calculated and stored.

Database:
  users-stores user details and their order
  sales–stores sales data
  commissions–stores commissions data
